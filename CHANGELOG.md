# Changelog

All notable changes to this project should be documented in this file.

## [0.1.0] - 2026-03-13

### Added

- Initial public beta release
- UI config flow for `username`, `password`, and `unit_id`
- Native Home Assistant polling with `DataUpdateCoordinator`
- Sensor entities for active Techem meters
- Reauth, reconfigure, diagnostics, and manual refresh support
- HACS metadata and integration branding assets
- Public automated tests and separate local beta validation workflow
